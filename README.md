# Micro Frontend Demo

## Setup

### Prereqs

1. Install [node](https://nodejs.org/)
   ```powershell
   choco install nodejs.install
   ```
2. Install [yarn](https://yarnpkg.org) package manager
   ```powershell
   choco install yarn
   ```

### Install dependencies

In the `microfrontend-demo` folder, run:

```powershell
yarn install
```

### Run application

```powershell
yarn start
```
